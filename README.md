# signupform

This is a basic multi-level sign up or registration form made using HTML, CSS and JavaScript.

### LANGUAGES USED :

HTML(HyperText Markup Language) - 40.1%
CSS(Cascading Style Sheets) - 35.4%
JS(JavaScript) - 24.5%
